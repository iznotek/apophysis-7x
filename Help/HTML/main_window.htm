Main window


The main window of Apophysis 7X.13
 

The main window is the first thing you will see when Apophysis starts. It consists of various areas which are explained below.

 

The menu- and toolbar area at the top
This area offers you full control of all Apophysis features. You can reach every other part of the program from there. Read the topics Main menu and Main toolbar to learn more about this area.

 

The batch listing on the left
All loaded flames in the current batch (collection of flames) are listed in the batch listing. You can use the batch as either a library of your fractal flames or as a collection of different versions of one fractal flame. All entries in this list are saved in a single file.

 

The main preview in the center of the window
This is the largest and most detailed preview panel in Apophysis. You can review your flame before rendering there. Use the quality drop-down in the toolbar to select the point density of the preview (often referenced as "Quality")
When the main preview is updating or a batch file loads, a progress bar will appear on the bottom of the main preview to tell you how far the current operation is done.

 

The status area at the bottom
You can read all important information there. From left to right, there are the following panels:

 

Loading status

If something went wrong during loading a batch you can see it there.

 

Elapsed and remaining times

These two panels show you the elapsed and remaining times of an operation. (like updating the main preview)

 

Selected flame name

This panel shows you the name of the selected flame within the currently loaded batch.

 

The main menu

 

Jump to: File - Edit - View - Flame - Variation - Script - Options - Help

 

 

File

 

New...

Depending on your settings, this entry either creates a blank flame or opens a window where you can select a basic flame style to start with.

 

Open...

Opens a batch of your choice.

 

Restore last autosave

Restores the last automatically saved flame. This only works when you enabled the autosave-feature in the options window.

 

Save parameters...

Saves the currently selected flame into a batch of your choice.

 

Save all parameters

Saves the currently opened batch into a new file.

 

Smooth palette...

Lets you select an image file from which the colors are read and applied to the currently selected flame.

 

Gradient browser...

Opens the gradient browser in which you can open and explore UltraFractal-formatted gradient files (.ugr) and select one of them to be applied to the currently selected flame.

 

Export UPR...

Exports the currently selected flame into the UltraFractal parameter format. (.upr) This only works with built-in variations applied to the flame and all 3D-related information will be lost.

 

Export flame...

Renders the currently selected flame using a renderer of your choice. Most users use flam3 as the export renderer of their choice. You can choose a renderer in the options window. Please remember that the export renderer may not support 3D-functionality or variations you used. Read the corresponding manual to learn more of the limitations and capabilities of your renderer.

 

Import GIMP-parameters

Lets you choose GIMP-parameters to be imported into Apophysis. This feature is deceprated in the Apophysis 7X-branch and may be removed in future versions.

 

Random batch

Generates a new batch of random flames. You can specify allowed variations, the batch size and the gradient limitations in the options window.

 

Exit

Closes Apophysis.

 

Edit

 

Undo

Reverts the recently made change within the currently selected flame.

 

Redo

Re-applies the recently undone change within the currently selected flame.

 

Save undo...

Saves the undo-buffer into a batch file. You can use this feature to store all recent changes within the currently selected flame into a single batch file.

 

Copy

Stores the Apophysis 2-formatted parameters of the currently selected flame into the clipboard.

 

Paste

Loads the contents of the clipboard as an Apophysis 2-formatted flame. If the clipboard does not contain properly formatted text data, this feature will not work.

 

Copy as UPR

Stores the UltraFractal-formatted parameters of the currently selected flame into the clipboard.

 

View

 

Full screen

Displays the currently selected flame on the entire screen.

 

Editor

Opens the editor window.

 

Adjust

Opens the camera tab of the adjustment window.

 

Gradient

Opens the gradient tab of the adjustment window.

 

Mutation

Opens the mutation window.

 

Image size

Opens the image size-tab of the adjustment window.

 

Flame

 

Reset Location

Resets the camera of the currently selected flame to its default location.

 

Randomize

Replaces the currently selected flame with a new random one.

 

Random Weights

Replaces all transform weights of the currently selected flame with random ones.

 

Equalize Weights

Equalizes all transform weights of the currently selected flame to a common value.

 

Compute Weights

Adjusts all transform weights of the currently selected flame to a value derived from the triangle area of the transform.

 

Calculate Color Values

Spreads the color values of all transforms in the currently selected flame evenly.

 

Randomize Color Values

Replaces all color values in the currently selected flame with random values.

 

Render to Disk...

Opens the rendering window for the currently selected flame.

 

Render All Flames

Opens the rendering window for all flames in the currently open batch.

 

Variation

 

Random

Mutates the currently selected flame using a random variation.

 

Built-in

Lists all built-in variations. Clicking one item in the sub-menu mutates the currently selected flame using the specified variation.

 

Plugins

Lists all variations provided by plugins. Clicking one item in the sub-menu mutates the currently selected flame using the specified variation.

 

Script

 

Run Script

Runs the currently selected script.

 

Stop Script

Stops the currently running script.

 

Open...

Opens a script from file.

 

Edit

Opens the script editor.

 

Manage Favorites

Opens the window where you can set favorite scripts to be displayed as entries within the script menu and having shortcut keys assigned.

 

Turn flame into script...

Generates a new script which reproduces the currently selected flame and opens the script editor. You can parameterize individual elements or add randomization within the script then.

 

Directory / More

Enumerates every script within the scripts folder which has not been selected as favorite. Clicking one item in the sub-menu opens and runs the specified script.

 

Options

 

Main Toolbar

Toggles the toolbar visibility.

 

Status Bar

Toggles the status bar visibility.

 

File Contents

Toggles the batch listing visibility.

 

Reset file content list width

Resets the width of the batch listing.

 

Options

Opens the options window.

 

Help

 

Contents

Opens this help file or a file you can specify in the options window.

 

The fractal flame algorithm

Opens a document which explains the scientific aspects of the fractal flame algorithm.

 

About

Opens a window which lists credits, version information and used third-party libraries.
