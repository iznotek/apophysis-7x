Transforms

Transforms are probably the most important elements of fractal flames. Each flame consists of a set of transforms where each transform consists of a set of properties (like weight, color and opacity) and a set of variations.

 

As the name says, transforms perform a transformation. In Apophysis, these transformations are affine - which means that moving, scaling, rotating and shearing is allowed. In a mathematic point of view, a transform is a two-by-three matrix which is multiplied by a vector. (which is the result of the sum of all variation functions within the transform). Apophysis tries to abstract this mathematical aspect by presenting the three columns of the matrix as the three points of a triangle. This way you are able to control the calculation visually. Moving, rotating or scaling a triangle has a similar effect to the transformation matrix.

 

A transform can take several properties:

 

Weight

The transform weight is a value which indicates how "important" the transform (and its variations) is. Higher values on one transform emphasizes the selected one where every other transform is faded. In Apophysis, the weight can range from almost-zero (0.000001) to 1,000. The default weight for each new transform is 0.5.

Additionally, Apophysis allows to define the weights in a more detailed way. This feature was named "Xaos" and you can learn more about it in the Xaos-topic.

 

Color

The transform color defines the color of the points generated by the corresponding transform. This must not be the exact color indicated. The finally selected color is also determined by the color speed parameter. To learn more about coloring your flames, read the coloring topic.

 

Color speed

The color speed is a parameter which supports the color distribution. The higher it is, the more the transform inherits colors from other transforms. In Apophysis, the color speed ranges from -1 (no color variation at all within the transform) to 1. (transform completely inherits other transforms colors) More about the color speed in the coloring topic.

 

Opacity

The transform opacity defines how much visible the points calculated by the corresponding transform are. In Apophysis, this value can range from zero (invisible) to one. (visible) As fractal flames consist of repeated shapes, the points of the corresponding transform may be copied by another transforms. These repeated shapes are not influenced by the opacity of the source transform but by the one of the transform which repeated the shapes.

 

Post-transform

A post-transform is an additional triangle which defines a transformation which is applied after the points are calculated. A the "normal" triangle can change the shape generated, the post-transform triangle does not. So this feature allows you to move, rotate, scale or shear the shape afterwards without changing its general shape.

 

In addition to normal transforms, Apophysis also supports one final transform to be applied. This special transform has influence over all transforms and its variations are applied to the overall shape generated by the other transforms
