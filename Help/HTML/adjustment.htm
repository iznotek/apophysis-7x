Adjustment window

The adjustment window

 

The adjustment window offers controls to change overall properties within the selected flame. From here you can change the camera settings, set a global color map and adjust the flame brightness and gamma values.

 

In the top section you will find a mini preview (clicking and dragging changes the 3D-properties of the camera - unlike the main preview which changes the 2D-properties) and a set of boxes where you can input the camera properties:

 

Depth blur

This field controls the optional depth-of-field effect you can set up for your 3D-flames. This only works properly if you adjusted pitch and yaw first. Setting it to zero disables the effect.

 

Pitch and Yaw

These two fields allow you to rotate the camera in the 3D space. If you set both to zero, the camera will look from top down to your flame.

 

Height

This field lets you adjust the vertical offset of the base plane of your flame. Positive values move it up, negative values move it down. By default, this offset is zero.

 

Perspective

With this field you can set the perspective distortion of your flame. It is commonly used to strengthen the 3D-effect of flames which where adjusted using pitch and yaw.

 

Scale

This field sets the point scale relative to the image size. It acts as a zoom feature but higher values will decrease the overall quality of the rendered image. If you want to preserve the quality, use the zoom feature instead. If you want to preserve the render speed, use this field.

 

 

The bottom section features a set of tabs which group several features into meaningful categories:

 

Camera

This tab contains the 2D-related camera controls.

The "zoom"-slider offers a zoom feature which preserves the flame quality but drastically decreases the render speed when using values greater than zero.

The X- and Y-position-sliders let you pan the camera after the 3D-transformations have been applied while the rotation-slider lets you rotate the camera after the values given in pitch and yaw have been applied.

 

Rendering

This tab contains fields which let you control the properties of image generation.

The gamma-slider lets you adjust the gamma-correction in order to improve the visibility of fine structures while the brightness-slider lets you adjust the overall brightness of the image. The vibrancy-slider lets you control the richness of the flames colors. You can also set the flame background color here.

The gamma treshold-box lets you fine-adjust the gamma correction in order to reduce the grain effect occuring on lower density areas. Higher values tend to make the image darker.

 

Gradient

The gradient tab offers you a powerful tool to create or adjust a gradient used as the main color map for your flame. On this tab, there is a black drop-down which provides a vast amount of built-in gradients. The slider above can take various modes which let you adjust the selected gradient. Click the text on the left of this slider to select the edit mode. With the text field on the right you can enter a value for the slider using your keyboard rather than adjusting it with your mouse using the slider. The button captioned "reset" restores the state of the gradient prior to opening the gradient tab or selecting an edit mode.

The top section of this tab contains a preview of the current gradient. Clicking and dragging on this section rotates the gradient even if you selected a different edit mode in the field below.

On the bottom right you will find a set of buttons:

 

Copies the current gradient into the clipboard (UltraFractal-formatted)

Pastes the current gradient from the clipboard. This does only work if there is a valid UltraFractal-formatted gradient in the clipboard. You may also copy a gradient from UltraFractal and paste it into Apophysis.

Opens the gradient browser in which you can open, save and explore a gradient file.

Lets you open an image and create a gradient from its colors.

 

Image size

This tab is grouped into three columns. The one on the left lets you enter an  size for the main preview, the one in the middle lets you manage up to three presets for the main preview sizes (click the wide button to load a preset or the button right next to it to save it) and the one on the right lets you apply your selected size. If the box on the right (captioned "resize main window") is not checked, the size image will be centered within the main preview area, elsewise the main window will be resized in order to match the preview size you entered.
