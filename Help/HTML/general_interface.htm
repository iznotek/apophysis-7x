Pre-defined procedures and constants

General procedures and functions

 

Preview;

Opens the preview window and displays the current flame.

 

print( 'string' );

Prints a string or variable to the debug box in the script editor.

 

ShowMessage( 'string' );

Opens a message box and displays the string to the user. The script will halt until the user pressed "OK".

 

ShowStatus( 'string' );

Sends a text message to the status bar in the main window.

 

InputQuery( 'title' , 'description' , variable );

Requires the user to input a value which is then stored into the variable at the third parameter. This is a function and it will return FALSE if the user pressed cancel.

 

ExtractFilePath( 'string' );

Returns a string containing the directory of the file given as the parameter.

 

CurrentFile;

Returns the path of the currently loaded batch file.

 

GetSaveFileName;

Opens a save dialog which requires the user to specify a file path. Returns an empty string if the user pressed cancel.

 

SetFlameFile( 'string' );

Loads a batch file into memory but does not list it in the batch list within the main window.

 

ListFile( 'string' );

Loads a batch file into memory but contrary to SetFlameFile, this will also list the flames into the batch list within the main window.

 

CopyFile( 'string' , 'string' , TRUE/FALSE);

Copies the file given with the first parameter to the target given by the second parameter. The third parameter determines whether to overwrite existing files.

 

SaveFlame( 'string' );

Saves a single flame into a batch file given as the parameter.

 

DeleteFile( 'string' );

Deletes a file from the users hard disk. WARNING: this does not move the file into the recycle bin but deletes it permanently. Use this procedure with caution.

 

Sin( number ); / Cos( number ); / Tan( number );

Returns the sine/cosine/tangent of a number in radians.

 

Random;

Returns a random number between zero and one.

 

Trunc( number );

Returns the integral part of a number.

 

Inc( variable ); / Dec( variable );

Increases or decreases a numeric variable by one.

 

StrToInt( integer ); / IntToStr( integer ); / FloatToStr( float ); / StrToFloat( string );

Converts an integer to a string, a float to a string or vice versa.

 

Clear;

Removes all transforms from the current flame.

 

Sqr( number ); / Sqrt( number );

Returns the square or the square root of the given number.

 

 

Constants and read-only variables

 

Transforms

The total number of transforms within the currently selected flame.

 

NXforms

The amount of allowed transforms per flame. This constant is 100 for normal Apophysis 7X-versions and 500 for the T500-versions.

 

NVars

The amount of loaded variations.

 

FileCount

The total number of flames within the currently loaded batch.

 

InstallPath

The path to the directory where Apophysis is located.

 

DateCode

The current date in the form of YYMMDD.

 

Stopped

Is TRUE when the user hit the stop button to halt a script.

 

Pi

The number PI (3.1415926535897932384626433832795...)
