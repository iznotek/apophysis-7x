Main toolbar

New

Depending on your settings, this entry either creates a blank flame or opens a window where you can select a basic flame style to start with.

 

Open

Opens a batch of your choice.

 

Save parameters

Saves the currently selected flame into a batch of your choice.

 

Render to Disk

Opens the rendering window for the currently selected flame.

 

Render All Flames

Opens the rendering window for all flames in the currently open batch.

 

View Batch as List

If toggled on, the batch listing will not contain thumbnails.

 

View Batch as Icons

If toggled on, the batch listing will contain thumbnails.

 

Undo

Reverts the recently made change within the currently selected flame.

 

Redo

Re-applies the recently undone change within the currently selected flame.

 

Reset Location

Resets the camera of the currently selected flame to its default location.

 

Full screen

Displays the currently selected flame on the entire screen.

 

Editor

Opens the editor window.

 

Adjust

Opens the camera tab of the adjustment window.

 

Gradient

Opens the gradient tab of the adjustment window.

 

Mutation

Opens the mutation window.

 

Image size

Opens the image size-tab of the adjustment window.

 

Parser output

Opens a window which shows all error messages generated by the last opened batch or selected flame.

 

Options

Opens the options window.

 

Show transparency

Toggles the visibility of flame transparency on or off. If toggled off, the selected background color of the flame will be shown as background.

 

Show guide lines

If toggled on, the flame in the main preview will be overlayed with guide lines.

 

Edit Script

Opens the script editor.

 

Run Script

Runs the currently selected script.

 

Stop Script

Stops the currently running script.

 

Move mode

If toggled on, clicking and dragging in the main preview will move the camera.

 

Rotate mode

If toggled on, clicking and dragging in the main preview will rotate the camera.

 

Zoom mode (zoom in)

If toggled on, clicking and dragging in the main preview will zoom the camera in.

 

Zoom mode (zoom out)

If toggled on, clicking and dragging in the main preview will zoom the camera out.
